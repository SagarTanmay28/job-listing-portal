# Job Listing Portal

A full-stack job listing portal connecting job seekers with employers.

![image alt](https://github.com/SagarTanmay28/job-listing-portal/blob/main/image1.png?raw=true)

## Features
- User authentication
- Job search and filters
- Resume upload
- Job applications
- Separate dashboards for employers and job seekers

## Tech Stack
- Node.js
- Express
- MongoDB
- EJS
- HTML, CSS, JavaScript

## Setup
1. Install dependencies
2. Configure .env
3. Run npm start
